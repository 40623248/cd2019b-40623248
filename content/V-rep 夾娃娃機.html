V-rep 夾娃娃機

onshape

Clip doll machine.ttt

在學校組裝vrep ，組裝好X、Y、Z、跟夾爪的軸。

----

回家後嘗試裝球體的軸(Spherical joints)，但是晃動得太嚴重，還會自轉，所以就先不裝了。

後來有把球體的軸改成水平跟垂直軸，讓他不會自轉，但還是會晃，感覺比較真實，載重量上做了很多的調整，才讓他看起來比較正常。(忘了錄影QQ)

-------------------------------------------------------------------------------------------------------

之後想讓夾娃娃機比較像真實的，就想讓他在抓起來的時候，回到原點並把爪子張開，將物體放開，於是我加了以下的程式。

---return
    result=sim.readProximitySensor(Sensor)
    if (result>0) then A=sim.getSimulationTime() end
    if (A > 0) then
        sliding_3 = sliding_3 +0.01
        if (sliding_3 >= 0) then sliding_x = sliding_x -0.01 end
        if (sliding_x <= -1.1) then sliding_2 = sliding_2 +0.01 end
        if (sliding_2 >= 1.3) then sliding_4 = -1 end
    end
    if (sliding_4 <= -0.02) then A=-1 end
---

解釋:在爪子內放一個球柱的感測器，感測到夾子夾起來，Z軸先上升到頂點，換X軸移動到指定的位子，再換Y軸移動到原點，最後將爪子鬆開。

程式最後一行是要讓 A 返回到 小於0 ， 不然就會無法控制其他軸。

function sysCall_init()
sliding_x = -1.1
sliding_2 = 1.3
sliding_3 = 0
sliding_4 = -0.04

A = 0

Sensor=sim.getObjectHandle("sensor")
X = sim.getObjectHandle('x_slider')
Y = sim.getObjectHandle('y_slider')
Z = sim.getObjectHandle('z_slider')
C = sim.getObjectHandle('catch_slider')

end


function sysCall_actuation()
sim.setJointTargetPosition(X,sliding_x)
sim.setJointTargetPosition(Y,sliding_2)
sim.setJointTargetPosition(Z,sliding_3)
sim.setJointTargetPosition(C,sliding_4)
---return
    result=sim.readProximitySensor(Sensor)
    if (result>0) then A=sim.getSimulationTime() end
    if (A > 0) then
        sliding_3 = sliding_3 +0.01
        if (sliding_3 >= 0) then sliding_x = sliding_x -0.01 end
        if (sliding_x <= -1.1) then sliding_2 = sliding_2 +0.01 end
        if (sliding_2 >= 1.3) then sliding_4 = -1 end
    end
    if (sliding_4 <= -0.02) then A=-1 end
---
message,auxiliaryData=sim.getSimulatorMessage()
        while message ~= -1 do
key=auxiliaryData[1]
sim.addStatusbarMessage('key:'..key)
            if (message==sim.message_keypress) then
--X_axis

                if (auxiliaryData[1]==100) then --a
                        -- 
                    if (sliding_x >= 1.2) then 
                    else sliding_x = sliding_x + 0.02
                    end
                end
                    
                if (auxiliaryData[1]==97) then --d
                        -- 
                    if (sliding_x <= -1.2) then 
                    else sliding_x = sliding_x - 0.02
                    end
                end
    ---axis y
                if (auxiliaryData[1]==115) then --w
                        -- 
                    if (sliding_2 >= 1.4) then 
                    else sliding_2 = sliding_2 + 0.02
                    end
                end
                if (auxiliaryData[1]==119) then --s
                        -- 
                    if (sliding_2 <= -1.4) then 
                    else sliding_2 = sliding_2 - 0.02
                    end
                end
    ---axis z
                if (auxiliaryData[1]==2007) then --up
                    if (sliding_3 >= 0.1) then 
                    else sliding_3 = sliding_3 + 0.05
                    end
                end
                if (auxiliaryData[1]==2008) then --down
                    if (sliding_3 <= -0.85) then 
                    else sliding_3 = sliding_3 - 0.05
                    end
                end
    ---catch
                if (auxiliaryData[1]==32) then --space
                    sliding_4 = 1
                end
            end
message,auxiliaryData=sim.getSimulatorMessage()
        end
end
